# PowerBi_Sales
This project presents an interactive Sales Dashboard built using Power BI to analyze and visualize sales performance across different dimensions such as salespersons, regions, categories, and time.
The dashboard provides a comprehensive overview of business performance, helping stakeholders make data-driven decisions through clear and interactive visualizations.

**Objectives:**
Monitor overall sales performance,
Analyze sales contribution by category and region,
Evaluate salesperson performance,
Track customer growth over time,
identify key trends and patterns in sales data.

**Key Insights:**
The Bars category contributes the highest share of total sales,
APAC region dominates in revenue generation compared to others,
sales performance varies significantly across different teams,
certain salespersons consistently outperform others,
customer growth shows variation across quarters.

**Results & Conclusion:**
The dashboard effectively transforms raw sales data into meaningful insights. It enables businesses to track performance, identify high-performing areas, and make strategic decisions to improve overall sales efficiency.
